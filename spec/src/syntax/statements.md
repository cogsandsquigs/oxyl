# Statements

> **<sup>Syntax:</sup>**\
> _Statement_:\
> &emsp; _Let_

Statements are immutable operations on data.

> **<sup>Syntax:</sup>**\
> _Let_:\
> &emsp; `"let"` _[Ident](./values_identifiers.md)_ `"="` _[Expression](./expressions.md)_ ( LINE_END | EOI )

This assigns some expression to the identifier in question.
